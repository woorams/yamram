# PRD (Product Requirements Document)

## 람얌이의 가계부

---

## 1. 프로젝트 개요

| 항목 | 내용 |
|------|------|
| **프로젝트명** | 람얌이의 가계부 |
| **목적** | 신혼부부의 수입과 지출을 체계적으로 관리하고, 복잡한 계좌 구조를 한눈에 파악할 수 있는 맞춤형 가계부 |
| **배경** | 기존 가계부 앱은 커스터마이징이 불가능하고, 신혼부부 특유의 복잡한 재정 구조(각자 용돈 통장, 개별 지출 계좌, 공동 생활비 통장 등)를 효과적으로 관리하기 어려움 |
| **타겟 사용자** | 신혼부부 2명 (우람, 윤미) - 비공개, 둘만 사용 |

---

## 2. 문제 정의

### 2.1 기존 가계부 앱의 한계
- 개별 커스터마이징 불가능
- 부부 공동 사용에 최적화되지 않음
- UI를 필요에 따라 변경할 수 없음
- 둘만의 프라이빗한 사용 환경 제공 불가

### 2.2 신혼부부 재정 관리의 어려움
- **복잡한 계좌 구조**: 각자 용돈 통장 + 각자 명의 지출 계좌 + 공동 생활비 통장
- **급여일 자금 배분 문제**: 월급이 들어오면 어느 계좌에 얼마를 넣어야 하는지 파악이 어려움
- **급여 통장 잔액 예측**: 이체될 금액을 예상하여 급여 통장에 얼마를 남겨둬야 하는지 계산이 복잡
- **지출 추적의 어려움**: 여러 계좌에서 발생하는 지출을 통합 관리하기 어려움

## 3. 핵심 기능 요구사항

### 3.1 수입/지출 입력

| 입력 항목 | 설명 | 필수 여부 |
|-----------|------|-----------|
| **금액** | 수입 또는 지출 금액 | 필수 |
| **구분** | 수입 / 지출 / 이체 | 필수 |
| **계좌** | 해당 거래가 발생한 계좌 | 필수 |
| **카테고리** | 지출/수입 분류 | 필수 |
| **날짜** | 거래 발생일 | 필수 |
| **메모** | 상세 내용 | 선택 |
| **입력자** | 우람 / 윤미 | 자동 |

- 항목은 추후 추가/삭제/수정 가능해야 함
- 한 명이 대표로 입력하는 방식

### 3.2 카테고리 체계

**지출 카테고리:**
| 카테고리 | 예시 |
|----------|------|
| 생활비 | 마트, 생필품, 일용품 |
| 식비 | 외식, 배달, 장보기 |
| 교통비 | 대중교통, 주유, 택시 |
| 주거비 | 월세, 관리비, 공과금 |
| 대출 상환 | 주택대출, 기타 대출 |
| 경조사비 | 축의금, 부의금, 선물 |
| 양가 부모님 | 용돈, 선물, 효도 관련 |
| 의료/건강 | 병원, 약국, 건강관리 |
| 쇼핑/의류 | 의류, 잡화 |
| 문화/여가 | 영화, 여행, 취미 |
| 보험 | 각종 보험료 |
| 통신비 | 휴대폰, 인터넷 |
| 구독료 | 넷플릭스, 유튜브 등 |
| 기타 | 분류 불가 항목 |

**수입 카테고리:**
| 카테고리 | 예시 |
|----------|------|
| 급여 (우람) | 월급 |
| 급여 (윤미) | 월급 |
| 부수입 | 이자, 환급 등 |
| 기타 수입 | 용돈, 선물 등 |

- 카테고리는 어드민에서 자유롭게 추가/수정/삭제 가능

### 3.3 계좌 관리

| 계좌 유형 | 설명 |
|-----------|------|
| 우람 급여 통장 | 월급 수령 계좌 |
| 윤미 급여 통장 | 월급 수령 계좌 |
| 우람 용돈 통장 | 개인 지출 계좌 |
| 윤미 용돈 통장 | 개인 지출 계좌 |
| 공동 생활비 통장 | 생활비 지출 전용 |
| 기타 계좌 | 저축, 비상금 등 |

- 계좌 추가/수정/삭제 가능
- 계좌별 잔액 추적
- **급여일 자금 배분표**: 급여 입금 시 각 계좌별 이체 금액 자동 계산/표시

### 3.4 대시보드 / 애널리틱스

**차트 및 통계:**
| 차트 | 설명 |
|------|------|
| 카테고리별 지출 비중 | 파이/도넛 차트 - 어디에 돈을 가장 많이 쓰는지 |
| 일별 지출 현황 | 바 차트 - 하루 단위 지출 추이 |
| 주별 지출 추이 | 라인 차트 - 주 단위 지출 흐름 |
| 월별 지출 추이 | 라인/바 차트 - 월 단위 지출 변화 |
| 수입 변동 그래프 | 라인 차트 - 수입의 월별 변화 |
| 예산 대비 지출 현황 | 프로그레스 바 - 카테고리별 예산 소진율 |
| 예산 한도별 현황 | 게이지 차트 - 전체 예산 대비 사용 현황 |

**핵심 지표 (한눈에 보기):**
- 이번 달 총 수입 / 총 지출 / 잔액
- 전월 대비 지출 증감
- 예산 소진율 (%)
- 가장 많이 쓴 카테고리 TOP 3

### 3.5 예산 관리

- 월별 전체 예산 설정
- 카테고리별 예산 한도 설정
- 각자(우람/윤미) 개인 생활비 예산 설정
- **예산 알림 기능**: 예산의 80%, 100% 도달 시 알림
- 예산 초과 시 경고 표시

### 3.6 급여일 자금 배분 기능

- 급여 금액 입력 시 각 계좌별 이체 금액 자동 계산
- 고정 지출(대출, 보험, 구독료 등) 자동 반영
- 급여 통장에 남겨둬야 할 최소 금액 표시
- 매월 배분 템플릿 설정 가능

### 3.7 데이터 일괄 업로드 (엑셀 / 구글 스프레드시트)

**업로드 기능:**
- 엑셀 파일(.xlsx) 업로드 시 거래 내역 일괄 등록
- 구글 스프레드시트 URL 연동 시 데이터 자동 가져오기
- 업로드 전 미리보기: 데이터 검증 후 오류 항목 표시 (잘못된 카테고리명, 누락 필드 등)
- 중복 데이터 감지 및 처리 옵션 (건너뛰기 / 덮어쓰기)
- 기존 데이터에 추가(append) 방식으로 반영

**양식 다운로드 기능:**
- 입력 양식 템플릿 다운로드 (엑셀 .xlsx 형식)
- 양식에 카테고리, 계좌 목록이 드롭다운으로 미리 세팅
- 양식에 입력 예시 데이터 포함

**업로드 양식 규격:**

| 열 | 항목 | 필수 | 예시 |
|----|------|------|------|
| A | 날짜 | 필수 | 2026-02-28 |
| B | 구분 | 필수 | 지출 / 수입 / 이체 |
| C | 금액 | 필수 | 32000 |
| D | 카테고리 | 필수 | 식비 |
| E | 계좌 | 필수 | 공동 생활비 통장 |
| F | 메모 | 선택 | 점심 식사 |

### 3.8 공동 사용

- 각자 계정으로 로그인 (우람, 윤미)
- 동일한 가계부 데이터 공유
- 둘만 접근 가능한 비공개 환경

## 4. 기술 스택

> 원칙: **무료**, 서버 비용 없음, 가볍고 심플하게

| 구분 | 기술 | 선정 이유 |
|------|------|-----------|
| **프론트엔드** | Next.js (React) | 웹앱 + PWA로 모바일 지원, 무료 배포 가능 |
| **UI 프레임워크** | Tailwind CSS | 빠른 UI 개발, 커스터마이징 용이 |
| **차트 라이브러리** | Chart.js 또는 Recharts | 무료, 다양한 차트 지원 |
| **백엔드** | Next.js API Routes | 별도 서버 불필요, 프론트와 통합 |
| **데이터베이스** | SQLite (로컬) 또는 Supabase (무료 티어) | 비용 없음, 간편한 설정 |
| **인증** | 간단한 비밀번호 인증 (2명만 사용) | 가벼운 인증 |
| **파일 처리** | SheetJS (xlsx) | 엑셀 파일 읽기/쓰기, 무료 오픈소스 |
| **구글 시트 연동** | Google Sheets API (무료) | 구글 스프레드시트 데이터 가져오기 |
| **배포** | Vercel (무료) 또는 로컬 서버 | 무료 호스팅 |
| **모바일 지원** | PWA (Progressive Web App) | 별도 앱 개발 불필요, 홈 화면 추가로 앱처럼 사용 |

**PWA 선택 이유:**
- 웹앱 하나로 PC + 모바일 동시 지원
- 앱스토어 등록 없이 홈 화면에 추가하여 앱처럼 사용
- 별도 모바일 앱 개발 비용/시간 절약

## 6. 비기능 요구사항

> 2명만 사용하는 프라이빗 서비스이므로 최소한으로 설정

- **인증**: 간단한 ID/PW 로그인 (2계정만 존재)
- **보안**: HTTPS 적용, 비밀번호 해싱
- **성능**: 특별한 요구사항 없음 (2명 사용)
- **백업**: 데이터 내보내기(CSV/Excel) 기능 제공

## 8. 화면 구성

### 8.1 페이지 목록

```
람얌이의 가계부
├── 로그인 페이지
├── 대시보드 (메인)
├── 입력 페이지
├── 내역 조회 (표)
├── 데이터 업로드
├── 차트/분석
├── 급여 배분 계산기
└── 설정 (어드민)
    ├── 카테고리 관리
    ├── 계좌 관리
    ├── 예산 설정
    └── 고정 지출 관리
```

### 8.2 각 페이지 핵심 요소

#### 로그인 페이지
- 심플한 로그인 폼 (ID, PW)
- "람얌이의 가계부" 로고/타이틀

```
┌─────────────────────────────────────┐

│ 총 수입   │ 총 지출   │ 잔액         │

│ [예산 소진율 프로그레스 바 ████░░ 64%] │

├────────────────────────────────────┤

```
